# 스킬: 트렌드 레이더 (Trend Radar)

## 목적
- 최신 LLM/에이전트 트렌드, 도구, MCP를 조사하여 파이프라인 개선 기회를 도출합니다.

## 입력
- 프로젝트 도메인/스코프
- 품질 목표(정확도, 안정성, 비용, 속도)

## 출력
- 적용 가능성 평가 목록
- 리스크/의존성/우선순위 정리

## 절차
1. **기술/툴 스캔**: 신규 모델, 프레임워크, MCP 서버 동향 조사.
2. **적용 가치 분석**: 비용 대비 효과, 도입 난이도, 위험도 평가.
3. **우선순위 정리**: 단기/중기/장기 로드맵 작성.
4. **결과 공유**: `reports/recommendations/`에 요약 기록.
