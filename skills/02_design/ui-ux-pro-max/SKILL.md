# UI/UX Pro Max Skill

이 스킬은 Antigravity의 **Architect** 및 **Developer** 에이전트가 사용자 중심의 뛰어난 디자인을 구현하도록 돕습니다.

## 1. 디자인 철학 (Design Philosophy)
- **Aesthetic Usability Effect**: 보기에 좋은 디자인이 사용하기도 쉽다고 느껴집니다.
- **Micro-Interactions**: 버튼, 입력창 등의 작은 상호작용에 애니메이션을 더해 생동감을 줍니다.
- **Accessibility (접근성)**: 모든 사용자가 차별 없이 사용할 수 있도록 명도 대비, 스크린 리더 지원을 고려합니다.

## 2. 디자인 시스템 (Design System)

### 2.1 색상 (Colors)
- **Primary**: 브랜드 아이덴티티를 나타내는 메인 컬러.
- **Secondary**: 보조 컬러 (강조, 알림 등).
- **Background**: 눈의 피로를 줄이는 다크 모드/라이트 모드 지원.
- **Semantic**: Success(Green), Error(Red), Warning(Yellow), Info(Blue).

### 2.2 타이포그래피 (Typography)
- **Hierarchy**: H1 > H2 > H3 > Body > Caption 순으로 명확한 위계를 설정합니다.
- **Readability**: 적절한 줄 간격(Line Height)과 자간(Letter Spacing)을 유지합니다 (예: 본문 줄 간격 1.5).

### 2.3 레이아웃 (Layout)
- **Grid System**: 12-column 그리드를 기본으로 반응형 웹을 설계합니다.
- **Spacing**: 4px 또는 8px 단위의 일정한 간격 시스템을 사용합니다 (8pt Grid).

## 3. UI 컴포넌트 가이드
- **Buttons**: 상태(Normal, Hover, Active, Disabled)에 따른 시각적 피드백 제공.
- **Inputs**: 명확한 레이블(Label)과 플레이스홀더(Placeholder), 에러 메시지 표시.
- **Cards**: 정보를 그룹화하여 보여주는 컨테이너, 그림자(Shadow)로 깊이감 표현.

## 4. 도구 통합
- 필요 시 `generate_image` 툴을 사용해 초기 디자인 목업을 생성할 수 있습니다.
